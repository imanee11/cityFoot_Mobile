# OasisSports

A Flutter mobile app for sports terrain booking and social features.

## Features
- Auth: Login / Register with Supabase Auth
- Home: Browse and filter sports terrains
- Reservations: Book a terrain, pick date/time/players, invite friends
- Friends: Add, accept/refuse/cancel friend requests
- Match Invitations: Accept or refuse match invitations
- Profile: View and edit your profile

## Setup
1. Run `flutter pub get`
2. Run `flutter run`

## Backend
Supabase — read/write only, no schema modifications.
